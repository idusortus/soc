# Soc Ops

Social games for mixers, coffee shops, and group gatherings.

## Games

- **Social Bingo** - Find people who match the squares and get 5 in a row
- **Two Truths & A Lie** - Guess which statement is false (coming soon)
- **Would You Rather** - Vote on dilemmas and debate (coming soon)
